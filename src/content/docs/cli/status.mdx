---
title: cropscode status
description: Show current user, worker, tenant, and connection state.
---

## Usage

```bash
cropscode status
```

## Description

Displays the current state of your CropsCode CLI setup at a glance:

```
User:         you@example.com
Worker:       registered (online)
Tenant:       your-tenant
Orchestrator: connected (wss://console.cropscode.io/ws/worker)
```

## Fields

| Field | Description |
|-------|-------------|
| User | Authenticated email (from `credentials.json`) |
| Worker | Registration status and online/offline state |
| Tenant | Which tenant this worker is assigned to |
| Orchestrator | Connection status and URL |

## Use Cases

- Verify you're authenticated after `cropscode login`
- Check worker connection after `cropscode worker`
- Debug connectivity issues
- Confirm tenant assignment
